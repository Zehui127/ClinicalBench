## 📈 使用建议

### 1. 分阶段测试
```
初级Agent → 测试v1.0（基础）
中级Agent → 测试v2.0（复杂）
高级Agent → 测试v3.0（真实场景）
```

### 2. 版本选择
- **教学/培训**: 使用v1.0，重点在基础问诊
- **常规评估**: 使用v2.0，增加区分度
- **高难度挑战**: 使用v3.0，测试真实场景能力

### 3. 逐步进阶
```
第一阶段: v1.0 → 通过后进入下一阶段
第二阶段: v2.0 → 通过后进入下一阶段
第三阶段: v3.0 → 最高难度
```
