1. **语言限制**: PrimeKG主要是英文，中文支持有限
2. **质量波动**: 生成的任务质量可能不稳定
3. **知识覆盖**: 受限于PrimeKG的覆盖范围
4. **首次加载**: 首次使用需要下载缓存数据
