## 推荐操作

### 1. 先测试小样本（一个CSV文件）

```bash
python -m DataQualityFiltering raw-data validate \
  --input "./data/raw/medical_dialogues/chinese_meddialog/Data_数据/IM_内科/内科5000-33000.csv" \
  --output validation_internal_medicine.json \
  --encoding gbk
```

**注意**：原始CSV文件可能是GBK编码，需要指定`--encoding gbk`

### 2. 查看验证报告

```bash
cat validation_internal_medicine.json
```

### 3. 根据报告决定是否继续转换

如果通过率 > 70%，说明数据质量较好，可以继续转换。
如果通过率 < 40%，说明数据质量较差，需要清理或寻找其他数据源。
